myFlix App - Documentation

This is my movie app where the user can find information about movies.

The table below describes each endpoint of this API and how to access it.

Description URL HTTP method Request body data format Response body data format
Get a list of all movies /movies GET None JSON object holding data about all movies. Structure:

{ Title: "Titanic",year:"1997", Description: "A seventeen-year-old aristocrat falls in love with a kind but poor artist aboard the luxurious,ill-fated R.M.S Titanic.", Genre: {Name: "Romance", Description: "romance is a narrative genre in literature that involves a mysterious, adventurous, or spiritual story line where the focus is on a quest that involves bravery and strong values, not always a love interest". }, Director: {Name: "James Cameron", Bio: "", Birth: "1954", Death: ""}, ImagePath:"https://www.imdb.com/title/tt0120338/mediaviewer/rm2647458304", featured: "false"}

Get data about a single movie by title /movies/[title] GET None JSON object holding data about a single movie. Structure same as above.
Get data about a movie genre /genre/[name] GET None JSON object holding data about a movie genre, containing name and description. Structure:

{Name: "Romance", Description: "romance is a narrative genre in literature that involves a mysterious, adventurous, or spiritual story line where the focus is on a quest that involves bravery and strong values, not always a love interest."}

Get data about a director /director/[name] GET None JSON object holding data about a director. Structure:

{Name: "James Cameron", Bio: "", Birth: "1954", Death: ""}

Add a new user to the registry. /users POST JSON object holding data about the user to add. Structure:

{name: "Zion", username: "ziontarakan", password: "pw456", email: "zion.tarakan@gmail.com", dateofbirth: "10/19/2015",favorites ["3"]}

JSON object holding data about the user that was add. Structure:

{ id: 0 ,name:"Zion" ,username: "zion tarakan", password: "pw456", email: "zion.tarakan@gmail.com", dateofbirth: "10/19/2015",favorites: ["3"]}

Update a user's information /users/[id] PUT JSON object holding data about the user to add. Structure:

{ name: "Zion",username: "zion tarakan", password: "pw456", email: "zion.tarakan@gmail.com", dateofbirth: "10/19/2015",favorites: ["3"]}

A message indicating the user's information was updated.
Add a movie to the user's favorites list /favorites/[id]/movies/[movieID] POST None A message indicating the movie title was added to the user's favorites list.
Delete a movie from the user's favorites list /favorites/[id]/movies/[movieID] DELETE None A message indicating the movie title was deleted from the user's favorites list.
Remove a user from the registry. /users/[id] DELETE None A message indicating the user was removed from the registry.
